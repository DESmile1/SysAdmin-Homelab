# SysAdmin Homelab

A Linux server environment built as a hands-on system administration practice project.

---

## Overview

This project is a continuously evolving Linux server setup focused on real-world system administration tasks such as networking, security, and service configuration.

The system is built incrementally, with each change documented after implementation.

---

## Current State

- OS: Ubuntu 25.10
- Hostname: srv01
- IP: 192.168.88.10
- Network: systemd-networkd (static configuration)
- SSH: active and reachable

## Recent Work
